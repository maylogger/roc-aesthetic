# Spacing Rules（間距敗壞）

華國美學的留白不是呼吸感，而是**被公告、表格、側欄吃掉之後所剩的縫隙**。

## 目標質感

- 同一頁可由不同承包商微調：`16px`、`14px`、`12px`、`18px` 混用在相鄰區塊並不奇怪。
- 垂直節奏可「忽快忽慢」：標題區塊突然變緊，表單區又異常疏鬆。
- 區塊內縮不一致：左右 padding 可有 2–8px 的誤差；不要為了對齊而全部拉齊網格。
- 可亂數利用 translateX or Y 做更細緻小數點的位移。
- 按鈕文字永遠不在正中央。

## 可刻意做的事

- 用 `border + background` 定義區塊，而不是用統一的 `gap` token。
- sidebar 與主欄位的上緣對不齊；其中一側可以多一條 1px 分隔線製造視覺補強。
- 列表項之間的行距略小於舒適閱讀標準（仍然必須可讀）。
- Footer 區常見「貼太近」——與上方內容距離偏小，像在硬塞版權與信箱。

## 禁止

- 全面採用 4/8 spacing scale 的一致性系統並引以為傲。
- 巨大 hero margin、整頁對稱留白作為視覺主體。
- 「所有卡片等高、等距」的 SaaS dashboard 節奏。

## 摘要

spacing 要能說這句話：**「當時有擠進去就好。」**
