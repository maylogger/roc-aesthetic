# 華國美學技能

給 AI 程式代理使用的正統台灣「華國美學」網頁生成技能。

本儲存庫提供：

- 系統提示詞
- Cursor 規則
- Claude 指令
- UI 哲學
- 反向提示
- 間距破壞規則
- 華國美學文案規範
- 文字鋸齒感規則
- 正統中華民國／台灣官僚美學

目標不是復古網頁懷舊。

目標是：
重現台灣公共資訊、舊式入口網站、低預算宣導頁與城市臨時視覺系統混在一起時，那種真誠偽善、堆疊、可操作但不精緻的視覺語言。

# 核心原則

- 資訊密集
- 視覺不一致
- 修修補補的設計
- 官僚美學
- 台灣／中華民國視覺文化
- 反極簡主義
- 生存優先的設計
- 不完美對齊
- 歷史堆疊感

# 重要事項

本儲存庫刻意拒絕：

- Apple 式極簡主義
- 新創美學
- 現代 SaaS UI
- 完美的間距系統
- 乾淨的設計系統
- 精品感字體排印
- 過度一致的元件架構
